# Introduction

open in new window    

This package provides set of ready to use Vue components and utilities for creating e-commerce applications using Heseya solution. Package is based on @heseya/store-core package/

# Instalation

TIP

You don't need to install @heseya/store-core package, it is already included in this package. But it can be useful if you want to use some of the core utilities.

You can install the package by running the following command:

yarn install @heseya/store-vue

# Config object

You can import and modify the config object that is being used in the package. This allows you to change the default settings for the SDK. They are mainly used by the @heseya/store-vue built-in components, but it is recommended to use them as well.

import { config } from '@heseya/store-vue'

config.formatAmount = (amount: number) => {
  return amount.toFixed(2)
}

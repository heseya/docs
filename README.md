# Heseya Documentation

[https://docs.heseya.com](https://docs.heseya.com)

## Development

Docs are made with [Vuepress v2](https://v2.vuepress.vuejs.org/).

```bash
$ yarn

# Run development server
$ yarn docs:dev
```

## Build

```bash
# Generate static files in `./docs/.vuepress/dist` directory
$ yarn docs:build
```
